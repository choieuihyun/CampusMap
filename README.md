# 프로젝트 명 : CampusMap

## 프로젝트 목적

  학교에 처음 방문하는 사람과 신입생, 재학생까지 포함하여 학교를 방문하는 사람들이 건물, 혹은 강의실을 찾는 것에 어려움을 겪지 않도록 하기 위함.
  
  그 외에도 교내 버스정류장과 공지사항을 추가하여 재학 중 필요한 정보를 편하게 얻을 수 있도록 함.
